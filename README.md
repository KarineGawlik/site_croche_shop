# Crochê Shop

Simple landing page developed for a crochet shop.

🔗 Live project: https://site-croche-shop.vercel.app/

## Technologies

- HTML
- CSS
- Vercel

## Purpose

This project was created to practice page structure, styling, layout organization and contact sections using HTML and CSS.

- Store presentation section
- Product images
- Contact form
- Social media area
- Address section
